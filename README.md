# Stopwatch Application

## Overview

A React-based stopwatch application that allows users to create and manage multiple stopwatches simultaneously.

## Features

- Create multiple independent stopwatch instances;
- Control each stopwatch individually;
- Reset stopwatch to zero;
- Remove unwanted stopwatch instances;
- Accurate time tracking.

## Installation & Setup

1. Clone the repository

```bash
  git clone https://github.com/RuMax21/L1-Hometask-Stopwatch.git
  cd L1-Hometask-Stopwatch
```

2. Install dependencies

```bash
  npm install
```

3. Start development server

```bash
  npm run dev
```

4. Build for production

```bash
  npm run build
```

## Usage

### Adding a stopwatch

- Click the "Add stopwatch" button to create a new stopwatch instance

### Controlling a stopwatch

- **Start**: Begin timing
- **Pause**: Pause the timer
- **Resume**: Continue from paused time
- **Clear**: Reset timer to zero & show "Start" button
- **Delete**: Remove the stopwatch completely
